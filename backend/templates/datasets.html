Corrected Dataset of Russian 1-grams from pre-revolutionary Google Books

Russian Google 1-grams up to 1918 year with an attemted correction of orthography supposedly mistaken by OCR. A fuller explanation is in the paper and the repository

File Format: Each of the files below is compressed tab-separated data. Each line has the following format: 

index raw_ngram TAB n_gram TAB year TAB match_count TAB volume_count TAB new_index TAB non_dictionary TAB corrected_ngram NEWLINE

As an example, here is the 1897th line from the a file of the Russian 1-grams (googlebooks-rus-corrected-1gram--a.gz): 

1  1верь 1верь 1897  1 1 1 True тверь

The rest information about the meanings of the columns' names you can read here.
