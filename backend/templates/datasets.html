Формат файлов: Каждый файл представляет собой данные разделенные табуляцией. Каждая срока аналогично нграмам Googl Books имеет следующий вид: 

index raw_ngram TAB n_gram TAB year TAB match_count TAB volume_count TAB new_index TAB non_dictionary TAB corrected_ngram NEWLINE

В качестве примера ниже привеодится 1897-я строка файла 1.tsv: 

1  1верь 1верь 1897  1 1 1 True тверь

Информацию о значениях названий колонок можно узнать на странице Googl Books.
